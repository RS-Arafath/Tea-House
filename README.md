# 🍵 Tea House — Tea Selling Landing Page

A modern and elegant landing page for a tea brand, built with HTML, CSS & JavaScript. Deployed on Vercel.

🔗 **Live Demo:** [https://tea-house-nine.vercel.app](https://tea-house-nine.vercel.app)

---

## 📸 Overview

Tea House is a beautifully designed static landing page for a tea and botanical solutions brand. It showcases featured tea products, brand story, client testimonials, and a newsletter subscription — all in a clean, minimal UI.

---

## ✨ Features

- **Hero Section** — Bold headline with "Explore More" CTA and a 5.00 Trustpilot rating badge
- **Featured Products** — Four product cards showcasing:
  - 🥛 Milk Tea
  - 🖤 Black Tea
  - 🍋 Lemon Tea
  - 🍃 Green Tea
- **Brand Story Section** — "Great Tea, Freshly Presented" with origin story inspired by Gong Cha (est. 2006, Taiwan)
- **Unique Taste & Premium Quality** highlights
- **Client Testimonials** — Testimonial slider with client names and roles
- **Newsletter Subscription** — Email capture in the footer
- **Footer** — Quick links, services, help section, and copyright

## 🚀 Getting Started

### Run Locally
```bash
# Clone the repository
git clone https://github.com/your-username/tea-house.git

# Navigate into the project
cd tea-house

# Open in browser
open index.html
```

No build tools or dependencies required — plain HTML/CSS/JS.

### Deploy on Vercel
```bash
npm install -g vercel
vercel
```
